# ROCO v0.1
Roco is a app that simplifies copy-pasting. It is a workflow which does copying,pasting and storing the text in an text file, open in a notepad.
At this current version, only available for Windows. 
* Download v0.1 exe https://github.com/roshanRobyn/roco/releases/tag/v.0.1
* # FEATURES
* Roco Button

    uses an overlay to show a button next to the selected text regardless of which application the text selected is present in.
* Notepad Paste Function

    the copied text is simultaneously pasted into a text file and opened in the notepad app. If the notepad app is not open already it will be opened on clicking the Roco button.
